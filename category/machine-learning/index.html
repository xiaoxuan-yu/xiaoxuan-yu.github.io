                          背景

                          Boltzmann 形式的分布

                          在广泛的条件下，统计力学所关心的微观状态 x 的平衡概率符合如下分布

                          x∼Cexp(−u(x))(1)

                          最为著名的例子就是 Boltzmann 分布。(1) 中的标量能量项 u(x) 包含系统的势能、温度和可选的其他热力学量。
