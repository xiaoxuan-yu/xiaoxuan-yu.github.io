                                                    # 10.A 迹

                                                    # 基的变更

                                                    • 定义: 单位矩阵,III 设nnn是正整数,n×nn\times nn×n对角矩阵:

                                                    (10⋱01) \begin{pmatrix} 1&&0\\ &\ddots&\\ 0&&1 \end{pmatrix} ​10​⋱​01​​

                                                    称为单位矩阵,记作III.

                                                    • 可逆的 逆 方阵AAA成为可逆的,如果存在一个同样大小的方阵BBB使得AB=BA=IAB=BA=IAB=BA=I,称BBB是AAA的逆,记作A−1A^{-1}A−1
                                                    • 线性映射之积的矩阵 假设u1,...,unu_1,...,u_nu1​,...,un​和v1,...,vnv_1,...,v_nv1​,...,vn​以及w1,...,wnw_1,...,w_nw1​,...,wn​都是VVV的基.设S,T∈L(V)S,T\in\mathcal L(V)S,T∈L(V).则:

                                                    M(ST,(u1,...,un),(w1,...,wn))=M(S,(v1,...,vn),(w1,...,wn))M(T,(u1,...,un),(v1,...,vn)) \begin{aligned} &\mathcal M(ST,(u_1,...,u_n),(w_1,...,w_n))=\\ &\mathcal M(S,(v_1,...,v_n),(w_1,...,w_n))\mathcal M(T,(u_1,...,u_n),(v_1,...,v_n)) \end{aligned} ​M(ST,(u1​,...,un​),(w1​,...,wn​))=M(S,(v1​,...,vn​),(w1​,...,wn​))M(T,(u1​,...,un​),(v1​,...,vn​))​

                                                    • 恒等算子关于两个基的矩阵 设u1,...,unu_1,...,u_nu1​,...,un​和v1,..,vnv_1,..,v_nv1​,..,vn​都是VVV的基,则矩阵M(I,(u1,...,un),(v1,...,vn))\mathcal M(I,(u_1,...,u_n),(v_1,...,v_n))M(I,(u1​,...,un​),(v1​,...,vn​))和M(I,(v1,...,vn),(u1,...,un))\mathcal M(I,(v_1,...,v_n),(u_1,...,u_n))M(I,(v1​,...,vn​),(u1​,...,un​))都是可逆的,且它们互为逆
                                                    • 基变更公式 设T∈L(V)T\in\mathcal L(V)T∈L(V),令u1,...,unu_1,...,u_nu1​,...,un​和v1,...,vnv_1,...,v_nv1​,...,vn​是VVV的基, A=M(I,(u1,...un),(v1,...,vn))A=\mathcal M(I,(u_1,...u_n),(v_1,...,v_n))A=M(I,(u1​,...un​),(v1​,...,vn​))则:

                                                    M(T,(u1,...,un))=A−1L(T,(v1,...,vn))A \mathcal M(T,(u_1,...,u_n))=A^{-1}\mathcal L(T,(v_1,...,v_n))A M(T,(u1​,...,un​))=A−1L(T,(v1​,...,vn​))A

                                                    # 迹:算子和矩阵间的联系

                                                    • 算子的迹 设T∈L(V)T\in\mathcal L(V)T∈L(V)
                                                      • 若F=C\bf F=CF=C,则TTT的迹等于TTT的按重数重复的全体本征值之和
                                                      • 若F=R\bf F=RF=R,则TTT的迹等于TCT_{\bf C}TC​的按重数重复的全体本征值之和

                                                    T的迹记作trace T\text{trace}\ Ttrace T

                                                    • 迹和特征多项式 设T∈L(V)T\in\mathcal L(V)T∈L(V),n=dim Vn=\text{dim}\ Vn=dim V则trace T\text{trace}\ Ttrace T等于TTT的特征多项式中zn−1z^{n-1}zn−1的系数的相反数
                                                    • 矩阵的迹 定义方阵AAA的迹为其对角线元素之和,记作trace A\text{trace}\ Atrace A
                                                    • ABABAB的迹等于BABABA的迹 如果AAA和BBB是相同阶数的方阵,那么trace(AB)=trace(BA)\text{trace}(AB)=\text{trace}(BA)trace(AB)=trace(BA)
                                                    • 算子的矩阵的迹不依赖于基 设T∈L(V)T\in\mathcal L(V)T∈L(V),如果u1,...,unu_1,...,u_nu1​,...,un​和v1,...,vnv_1,...,v_nv1​,...,vn​是VVV的基,则

                                                    trace M(T,(u1,...,un))=trace M(T,(v1,...,vn)) \text{trace}\ \mathcal M(T,(u_1,...,u_n))=\text{trace}\ \mathcal M(T,(v_1,...,v_n)) trace M(T,(u1​,...,un​))=trace M(T,(v1​,...,vn​))

                                                    • 算子的迹等于矩阵的迹 若T∈L(V)T\in\mathcal L(V)T∈L(V),则trace T=trace M(T)\text{trace}\ T=\text{trace}\ \mathcal M(T)trace T=trace M(T)
                                                    • 迹是可加的 若S,T∈L(V)S,T\in\mathcal L(V)S,T∈L(V),则trace(S+T)=trace S+trace T\text{trace}(S+T)=\text{trace}\ S+\text{trace}\ Ttrace(S+T)=trace S+trace T
                                                    • 恒等算子不是STSTST和TSTSTS的差 不存在算子S,T∈L(V)S,T\in\mathcal L(V)S,T∈L(V)使得ST−TS=IST-TS=IST−TS=I

                                                    # 10.B 行列式

                                                    # 算子的行列式

                                                    • 定义:算子的行列式 设T∈L(V)T\in\mathcal L(V)T∈L(V)
                                                      • 若F=C\bf F=CF=C,则TTT的行列式是TTT的按重数重复的全体本征值之积
                                                      • 若E=R\bf E=RE=R,则TTT的行列式是TCT_{\bf C}TC​的按重数重复的全体本征值之积

                                                    T的行列式记作det T\text{det}\ Tdet T.

                                                    • 行列式和特征多项式 设T∈L(V), n=dim VT\in\mathcal L(V),\ n=\text{dim}\ VT∈L(V), n=dim V,则det T\text{det}\ Tdet T等于(−1)n(-1)^n(−1)n乘以TTT的特征多项式的常数项
                                                    • 特征多项式、迹和行列式 设T∈L(V)T\in\mathcal L(V)T∈L(V).则TTT的特征多项式可以写作zn−(trace T)zn−1+⋯+(−1)n(det T)z^n-(\text{trace\ }T)z^{n-1}+\cdots+(-1)^n(\text{det\ }T)zn−(trace T)zn−1+⋯+(−1)n(det T)
                                                    • 可逆等价于行列式非0VVV上的算子是可逆的当且仅当它的行列式是非零的
                                                    • TTT的特征多项式等于det(zI−T)\text{det}(zI-T)det(zI−T)

                                                    # 矩阵的行列式

                                                    • 排列 (1,...,n)(1,...,n)(1,...,n)的一个排列是一个组(m1,...,mn)(m_1,...,m_n)(m1​,...,mn​),1,...,n1,...,n1,...,n中的每个数恰好在其中出现一次 (1,...,n)(1,...,n)(1,...,n)中的所有排列组成的集合记为perm n\text{perm}\ nperm n
                                                    • 排列的符号
                                                      • 如果在组(m1,...,mn)(m_1,...,m_n)(m1​,...,mn​)中使得1≤j<k≤n1\leq j<k\leq n1≤j<k≤n且jjj出现在kkk之后的整数对(j,k)(j,k)(j,k)的个数为整数,那么排列的符号定义为111,反之定义为−1-1−1
                                                      • 也就是说,如果自然顺序被改变偶数次,排列的符号等于111;反之等于−1-1−1
                                                    • 矩阵的行列式 n×nn\times nn×n矩阵

                                                    (A1,1⋯A1,n⋮⋮An,1⋯An,n) \begin{pmatrix} A_{1,1}&\cdots&A_{1,n}\\ \vdots&&\vdots\\ A_{n,1}&\cdots&A_{n,n} \end{pmatrix} ​A1,1​⋮An,1​​⋯⋯​A1,n​⋮An,n​​​

                                                    的行列式定义为:

                                                    det A = ∑(m1,...,mn)∈perm n(sign(m1,...mn))Am1,n⋯Amn,n \text{det}\ A\ =\ \sum_{(m_1,...,m_n)\in\text{perm}\ n}(\text{sign}(m_1,...m_n))A_{m_1,n}\cdots A_{m_n,n} det A = (m1​,...,mn​)∈perm n∑​(sign(m1​,...mn​))Am1​,n​⋯Amn​,n​

                                                    • 交换矩阵的两列 AAA和BBB是通过交换两列得到的矩阵,则det A = det B\text{det}\ A\ =\ \text{det}\ Bdet A = det B
                                                    • 具有两相同列的矩阵 若AAA有两列值相同的,则det A=0\text{det}\ A=0det A=0
                                                    • 重排矩阵的列 设A = (A⋅,1,...,A⋅,n)A\ =\ (A_{\cdot,1},...,A_{\cdot,n})A = (A⋅,1​,...,A⋅,n​)是n×nn\times nn×n矩阵,(m1,...mn)(m_1,...m_n)(m1​,...mn​)是一个排列,那么:

                                                    det(A⋅,1,...,A⋅,n)=(sign(m1,...,mn))det A \text{det}(A_{\cdot,1},...,A_{\cdot,n})=(\text{sign}(m_1,...,m_n))\text{det}\ A det(A⋅,1​,...,A⋅,n​)=(sign(m1​,...,mn​))det A

                                                    • 行列式是每一列的线性函数 设k,nk,nk,n是满足1≤k≤n1\leq k\leq n1≤k≤n的正整数,固定除了A⋅,kA_{\cdot,k}A⋅,k​之外的那些n×1n\times 1n×1矩阵A⋅,1...A_{\cdot,1}...A⋅,1​...,那么把A⋅,kA_{\cdot,k}A⋅,k​映射为:

                                                    det(A⋅,1,...,A⋅,k,...,A⋅,n) \text{det}(A_{\cdot,1},...,A_{\cdot,k},...,A_{\cdot,n}) det(A⋅,1​,...,A⋅,k​,...,A⋅,n​)

                                                    的函数,是从F\bf FF上的n×1n\times 1n×1矩阵构成的向量空间到F\bf FF的线性映射

                                                    • 行列式是可乘的 若AAA和BBB是大小相同的方阵,则det(AB) = det(BA) = det(A)det(B)\text{det}(AB)\ =\ \text{det}(BA)\ =\ \text{det}(A)\text{det}(B)det(AB) = det(BA) = det(A)det(B)
                                                    • 算子的行列式并不依赖于基,且等于其矩阵的行列式
                                                    • 算子的行列式是可乘的 设S,T∈L(V)S,T\in\mathcal L(V)S,T∈L(V),则det(ST) = det(TS) = det(S)det(T)\text{det}(ST)\ =\ \text{det}(TS)\ =\ \text{det}(S)\text{det}(T)det(ST) = det(TS) = det(S)det(T)
