                                                          # 9.A 复化

                                                          # 向量空间的复化

                                                          • VVV的复化VCV_{\bf C}VC​
                                                            设VVV是实向量空间.
                                                            • VVV的复化等于V×VV\times VV×V,其元素是有序对(u,v)(u,v)(u,v).其中u,v∈Vu,v\in Vu,v∈V,但我们写作u+ivu+\text{i}vu+iv
                                                            • 定义VCV_{\bf C}VC​上的加法为

                                                            (u1+iv1)+(u2+iv2)=(u1+u2)+i(v1+v2) (u_1+\text{i}v_1)+(u_2+\text{i}v_2)=(u_1+u_2)+\text{i}(v_1+v_2) (u1​+iv1​)+(u2​+iv2​)=(u1​+u2​)+i(v1​+v2​)

                                                            • 定义VCV_{\bf C}VC​上的复标量乘法为

                                                            (a+bi)(c+di)=(au−bv)+i(av+bu) (a+b\text{i})(c+d\text{i})=(au-bv)+\text{i}(av+bu) (a+bi)(c+di)=(au−bv)+i(av+bu)

                                                            • 根据上述加法和标量乘法的定义,VCV_{\bf C}VC​是复向量空间
                                                          • VCV_{\bf C}VC​的基
                                                            设VVV是实向量空间.
                                                            • 如果v1,...,vnv_1,...,v_nv1​,...,vn​是VVV的基,则它也是VCV_{\bf C}VC​的基
                                                            • VCV_{\bf C}VC​的维数等于VVV的维数

                                                          # 算子的复化

                                                          • TTT的复化,TCT_{\bf C}TC​
                                                            设VVV是实向量空间,T∈L(V)T\in\mathcal L(V)T∈L(V).TTT的复化是定义为TC(u+iv)=Tu+iTvT_{\bf C}(u+\text{i}v)=Tu+\text{i} TvTC​(u+iv)=Tu+iTv的算子TC∈L(VC)T_{\bf C}\in\mathcal{L}(V_{\bf C})TC​∈L(VC​),其中u,v∈Vu,v\in Vu,v∈V
                                                          • TCT_{\bf C}TC​的矩阵等于TTT的矩阵
                                                            设V1,...,vnV_1,...,v_nV1​,...,vn​是实向量空间VVV的基,T∈L(V)T\in\mathcal L(V)T∈L(V)则M(T) =M(TC)\mathcal M(T)\ =\mathcal M(T_{\bf C})M(T) =M(TC​),其中这两个矩阵都是关于基v1,...,vnv_1,...,v_nv1​,...,vn​的矩阵
                                                          • 每个算子都具有一维或者二维不变子空间

                                                          proof:
                                                          非零的有限维复向量空间上的每个算子都具有本征值,从而具有以为不变子空间.
                                                          因此假设VVV是实向量空间,T∈L(V)T\in\mathcal L(V)T∈L(V).复化TCT_{\bf C}TC​有本征值a+bia+b\text{i}a+bi,其中a,b∈Ra,b\in \bf Ra,b∈R.因此存在不全为000的u,v∈Vu,v\in Vu,v∈V使得TC(u+iv) = (a+bi)(u+vi)T_{\bf C}(u+\text{i} v)\ =\ (a+b\text{i})(u+v\text{i})TC​(u+iv) = (a+bi)(u+vi),也就是说:

                                                          Tu+iTv=(au−bv)+(av+bu)i Tu+\text{i}Tv=(au-bv)+(av+bu)\text{i} Tu+iTv=(au−bv)+(av+bu)i

                                                          于是:

                                                          Tu=au−bv, Tv=av+bu Tu=au-bv,\ Tv=av+bu Tu=au−bv, Tv=av+bu

                                                          令UUU等于组u,vu,vu,v在VVV中的张成,那么UUU是VVV的一维或二维不变子空间.

                                                          # 复化的极小多项式

                                                          • TCT_{\bf C}TC​的极小多项式等于TTT的极小多项式
                                                            设VVV是实向量空间,T∈L(V)T\in\mathcal L(V)T∈L(V).则TCT_{\bf C}TC​的极小多项式等于TTT的极小多项式

                                                          # 复化的本征值

                                                          • TCT_{\bf C}TC​的是本征值,T∈L(V), λ∈RT\in\mathcal L(V),\ \lambda\in \bf RT∈L(V), λ∈R则λ\lambdaλ是TCT_{\bf C}TC​的本征值当且仅当λ\lambdaλ是TTT的本征值
                                                          • TC−λIT_{\bf C}-\lambda ITC​−λI和TC−λˉIT_{\bf C}-\bar\lambda ITC​−λˉI
                                                            设VVV是实向量空间,T∈L(V)T\in\mathcal L(V)T∈L(V),λ∈C\lambda\in\bf Cλ∈C,jjj是非负整数,u,v∈Vu,v\in Vu,v∈V,则:

                                                          (TC−λI)j(u+iv)=0  ⟺  (TC−λˉI)j(u−iv)=0 (T_{\bf C}-\lambda I)^j(u+\text{i}v)=0\iff (T_{\bf C}-\bar\lambda I)^j(u-\text{i}v)=0 (TC​−λI)j(u+iv)=0⟺(TC​−λˉI)j(u−iv)=0

                                                          推论:TCT_{\bf C}TC​的非实数的本征值成对出现

                                                          • λ\lambdaλ的重数等于λˉ\bar\lambdaλˉ的重数
                                                            设VVV是实向量空间,T∈L(V)T\in\mathcal L(V)T∈L(V),λ∈C\lambda\in \bf Cλ∈C是TCT_{\bf C}TC​的本征值.则λ\lambdaλ作为TCT_{\bf C}TC​的本征值等于λˉ\bar\lambdaλˉ作为TCT_{\bf C}TC​的本征值的重数
                                                          • 以上若干定理所能给出的重要结果是奇数维向量空间上的每个算子都有本征值

                                                          # 复化的特征多项式

                                                          • TCT_{\bf C}TC​的特征多项式
                                                            设VVV是实向量空间,T∈L(V)T\in\mathcal L(V)T∈L(V).则TCT_{\bf C}TC​的特征多项式的系数都是实数.
                                                          • 特征多项式
                                                            设VVV是实向量空间,T∈L(V)T\in\mathcal L(V)T∈L(V).则TTT的特征多项式定义为TCT_{\bf C}TC​的特征多项式
                                                          • 特征多项式的次数和零点
                                                            设VVV是实向量空间,T∈L(V)T\in\mathcal L(V)T∈L(V),则:
                                                            • TTT的特征多项式的系数都是实的

                                                            • TTT的所有本征值恰为TTT的特征多项式的所有实零点

                                                            设T∈L(V)T\in\mathcal L(V)T∈L(V),qqq是TTT的特征多项式,则q(T)=0q(T)=0q(T)=0
                                                          • 特征多项式是极小多项式的多项式倍

                                                            • TTT的特征多项式是TTT的极小多项式的多项式倍
